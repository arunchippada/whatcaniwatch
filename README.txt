Dependencies
------------

The following dependencies need to be installed to compile and run the sources

1) Apache
2) PHP (including HTTP_Request2 module)
3) PHP Sesame (https://github.com/alexlatchford/phpSesame)
4) Facebook PHP sdk (https://github.com/facebook/facebook-php-sdk/archive/master.zip)
5) Java (http://java.sun.com/javase/downloads/index.jsp)
6) Sesame (http://sourceforge.net/projects/sesame/files/latest/download)
7) Jackson (http://wiki.fasterxml.com/JacksonDownload)
8) Jsoup (http://jsoup.org/download)
9) Perl
